# Changelog

## 0.1.0 - 2026-06-07

Initial public-candidate scaffold.

- Added dependency-light SRT parsing.
- Added slide Markdown parsing with `<!-- slide_idx: N -->` markers.
- Added deterministic slide-to-cue alignment.
- Added CLI JSON export.
- Added synthetic lecture fixture.
- Added tests, GitHub Actions CI, MIT license, security policy, and contribution guide.

This release candidate intentionally excludes private lecture materials,
credential handling, course downloaders, copyrighted decks, and medical exam
content.
