# Roadmap

## 0.1.x

- Keep the public core dependency-light.
- Add more synthetic SRT edge-case fixtures.
- Add JSON schema validation for alignment output.
- Add issue templates for bug reports and fixture requests.

## 0.2.x

- Add optional slide text extraction adapters.
- Add a richer deterministic scoring model.
- Add fixture anonymization guidance.

## Non-goals

- No private course workflow automation.
- No credential handling.
- No medical, legal, or clinical advice.
- No automatic summarization pipeline in the core package.
